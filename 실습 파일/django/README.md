### clone

> 최초에 한번 본 원격 저장소를 복제하기 위해 사용합니다.

```bash
$ git clone https://lab.ssafy.com/06/class-a/django
```

### pull

> 본 저장소에 업데이트가 된 내용을 받아오기 위해 사용합니다.

```bash
$ git pull origin master
```
